# Code for running tests and generating figures in submission to CPC

See the FaADE package here: [github](https://github.com/Spiffmeister/FaADE.jl)

See the arxiv submission for details: [arxiv:](https://arxiv.org/abs/2604.20766)

Tests are run with the convergence branch of the code, which allows for a fixed number of time steps to be run.

```bash
julia --project
```

then run,

```julia
]instantiate
```

to install dependences. Data or figures can then be generated by running

```bash
julia --project=. <FILENAME>
```

Note that convergence plots are embedded in the paper with Tikz using PGFPlots, the scripts here will generate the data for the convergence figures only.
